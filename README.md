# 🎮 AI Player Behaviour Analyzer

A Deep Learning-based Game Analytics System that predicts player behaviour, stress risk, and disengagement trends using LSTM networks.

## 🚀 Project Summary 

Designed and deployed an end-to-end AI analytics framework that transforms raw gameplay logs into actionable insights. The system classsifies players into Casual, Regular, and Addicted categories while predicting Stress Risk and Reduced Usage Risk in real time.

## 🧠 Key Highlights

- Build LSTM-based multi-output deep learning models.
- Engineered behavioural features such as Engagement Score and Activity Consistency
- Developed an interactive Streamlit dashboard for real-time analysis
- Implemented risk leaderboard, distribution charts, and downloadable reports
- Designed the system to detect early churn and unhealthy gaming patterns

## Tech Stack
Python | Tensorflow | Keras | Scikit-learn | Pandas | Streamlit | Plotly 

## Impact
- Enables early detection of player churn and burnout
- Supports responsible game design decisions
- Converts gameplay data into strategic retention insights
